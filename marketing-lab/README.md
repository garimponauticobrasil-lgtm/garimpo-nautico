# Marketing Lab

Area local para testar tecnicas de marketing antes de levar para o site principal.

Regra do projeto: toda tecnica, campanha, promessa, oferta, CTA, landing page ou teste de marketing deve nascer nesta pasta primeiro. Depois de testado localmente e aprovado, a parte que funcionou pode ser levada para o site principal e publicada.

Rota local:

```txt
http://127.0.0.1:5173/marketing-lab
```

Esta pasta e local para experimento. Nao leve campanhas para a home sem aprovar antes.

Fluxo obrigatorio:

1. Criar ou ajustar uma hipotese nesta pasta.
2. Testar visualmente e validar a mensagem.
3. Copiar so o que funcionar para `src/`.
4. Publicar no GitHub Pages depois da aprovacao.
