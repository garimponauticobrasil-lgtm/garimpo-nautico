# Marketing Lab

Área local para testar técnicas de marketing antes de levar para o site principal.

Regra do projeto: toda técnica, campanha, promessa, oferta, CTA, landing page ou teste de marketing deve nascer nesta pasta primeiro. Depois de testado localmente e aprovado, a parte que funcionou pode ser levada para o site principal e publicada.

Rota local:

```txt
http://127.0.0.1:5173/marketing-lab
```

Esta pasta está no `.vercelignore`, então não é publicada na Vercel.

Fluxo obrigatório:

1. Criar ou ajustar uma hipótese nesta pasta.
2. Testar visualmente e validar a mensagem.
3. Copiar só o que funcionar para `src/`.
4. Publicar na Vercel depois da aprovação.
